# Инструкция по установке VPK через ArdysaModsTools

После того как вы создали свой оптимизированный `pak01_dir.vpk`, выполните следующие действия:

1. **Импорт архива:**
   - Откройте приложение **ArdysaModsTools**.
   - Нажмите кнопку **Install mod pack**.
   - Выберите вариант **Manual install**.
   - Выберите ваш созданный файл `pak01_dir.vpk`.
   - **Важно:** При запросе выберите "оставить оригинал" (Keep original).
   - Нажмите **Continue** и дождитесь завершения установки.

2. **Применение модов на окружение:**
   - Перейдите во вкладку **MISCELLANEOUS**.
   - Выберите нужные моды (вышки, ландшафт, погодные эффекты).
   - Нажмите кнопку **Generate**.
   - Выберите **Add to Current Mods** (это применит модификации поверх уже установленного VPK).

3. **Проверка целостности (Финализация):**
   - Перейдите в раздел **Patch update**.
   - Нажмите **Verify mod file**.
   - После завершения проверки нажмите **Patch Update**.

Теперь ваш мод полностью готов к работе в игре!
